## Steps to install

Install Dependencies
- [Docker](https://docs.docker.com/engine/install/)
- [Docker Compose](https://docs.docker.com/compose/install/)

Clone the repo
```
git clone https://github.com/DevGamer9991/Profile-Website.git
```

Build the container
```
docker-compose build
```

Run the container

```
docker-compose up -d
```
